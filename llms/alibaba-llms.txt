# Alibaba

> Alibaba is a multinational technology conglomerate founded in 1999. Alibaba Cloud (Aliyun) is a global leader in cloud computing and AI, exposing hundreds of services through a comprehensive API ecosystem. The OpenAPI Explorer is a web interface for discovering, testing, and generating SDK code for Alibaba Cloud service APIs across Java, Python, Node.js, PHP, Go, and more. Authentication uses Alibaba Cloud AccessKey (HMAC signature) credentials.

This llms.txt was generated by the API Evangelist enrichment pipeline from apis.yml and repo artifacts; Alibaba does not publish an llms.txt on its docs host.

## APIs

- [Alibaba Cloud API](https://api.alibabacloud.com/): Comprehensive API ecosystem covering ECS, OSS, ACK, RDS, MaxCompute, Function Compute, API Gateway, CDN, and hundreds of additional cloud services. Base URL https://ecs.aliyuncs.com (per-service *.aliyuncs.com endpoints).
